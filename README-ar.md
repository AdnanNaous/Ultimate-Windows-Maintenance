## 📝 الوصف
أداة الصيانة الشاملة هي أداة برمجية آمنة واحترافية للحفاظ على نظام ويندوز 11 الخاص بك في أفضل أداء ممكن. تم تصميم الأداة بأسلوب برمجي دفاعي صارم لضمان عدم حدوث أي أخطاء، مع تجنب تعديلات الريجستري (Registry) الخطيرة، وإدارة دقيقة للعمليات.

## ✨ المميزات
- **فحص صحة النظام:** التحقق من إصدار النظام، صحة الأقراص (SMART)، حالة الذاكرة، والأخطاء الحرجة في سجل الأحداث.
- **تنظيف النظام:** تنظيف الملفات المؤقتة بشكل آمن، ذاكرة التخزين المؤقت لتحديثات ويندوز، ذاكرة DirectX، وإفراغ سلة المهملات بذكاء وتجاوز الملفات المقفلة.
- **إصلاح النظام:** أتمتة أوامر الإصلاح العميقة (`sfc /scannow` و `DISM RestoreHealth`) وتنظيف متجر المكونات (Component Store) مع إدارة صارمة لوقت التشغيل.
- **التحديثات:** تشغيل تحديثات ويندوز، ترقية البرامج باستخدام `winget`، وتحديث إضافات PowerShell.
- **التعريفات:** اكتشاف تعريفات AMD و NVIDIA بأمان تام دون حذفها أو فرض تثبيتها.
- **الأمان:** تحديث قواعد بيانات Microsoft Defender، إجراء فحص بالخلفية، وعزل التهديدات النشطة فوراً.
- **التوافق مع G-Helper:** اكتشاف حالة تطبيق G-Helper وعمل نسخة احتياطية آمنة لإعداداته.
- **التقارير:** إنشاء تقارير شاملة بعد كل عملية صيانة بصيغ JSON و TXT و HTML.

## 📚 دليل الاستخدام
### المتطلبات
- **نظام التشغيل:** ويندوز 11
- **الطرفية:** PowerShell 7 أو أحدث
- **الصلاحيات:** يجب تشغيل الطرفية بصلاحيات المسؤول (Administrator).

### طريقة التشغيل
1. افتح **PowerShell 7** بصلاحيات المسؤول (Run as Administrator).
2. قم بتحميل أو استنساخ المشروع إلى جهازك.
3. انتقل إلى مجلد الأداة من خلال سطر الأوامر.
4. قم بتشغيل السكربت عبر الأمر التالي:
   ```powershell
   .\Start-Maintenance.ps1
   ```

> [!WARNING]
> **تحذير هام:** لا تقم بفتح ملف `.ps1` ونسخ الكود ولصقه مباشرة داخل نافذة PowerShell. يجب تشغيل السكربت كملف باستخدام الأمر أعلاه حتى يتمكن من العثور على الإعدادات والمجلدات الخاصة به!

*(ملاحظة: إذا أردت تخطي إنشاء نقطة استعادة النظام لتوفير الوقت، يمكنك استخدام الأمر `-SkipRestorePoint`)*
